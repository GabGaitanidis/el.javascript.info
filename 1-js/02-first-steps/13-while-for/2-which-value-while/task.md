# Ποιες τιμές δείχνει το while loop?

Για κάθε επανάληψη βρόχου, γράψτε ποια τιμή εξέρχεται και, στη συνέχεια, συγκρίνετε τη με τη λύση. 

Και οι δύο βρόχοι  `alert` τις ίδιες τιμές, ή όχι;

1. Η μορφή prefix `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. H μορφή postfix `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
